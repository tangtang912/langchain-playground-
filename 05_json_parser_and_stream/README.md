# JSON 解析器与流式传输

本练习展示了两个高级特性：
1. **JsonOutputParser**：将 LLM 输出的 JSON 字符串自动解析为 Python 字典，方便后续代码直接使用。
2. **stream()**：流式输出，模拟"打字机效果"，在生成长文本时能显著提升用户体验。

## 链式调用逻辑
1. 生成名字（要求输出 JSON） 
2. 解析 JSON 提取 `name` 字段
3. 把 `name` 填入第二个提示词
4. 让模型解析名字含义
5. 逐字流式打印结果

## 运行
`python main.py`

## 预期效果
你会看到"张薇"的含义解释以逐字方式打印出来。
